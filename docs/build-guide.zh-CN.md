# 装配指南

*[English](build-guide.md) | 中文*

下面是小车的装配与调试顺序。元件清单见 [../hardware/bom.md](../hardware/bom.md)，
电路见 [../hardware/schematic.svg](../hardware/schematic.svg)。

## 1. 电路板与连线

信号通路上的元件都焊在主板上：LM393 芯片座、三极管、电阻、LED，还有传感器、电机、电池盒
和开关的引线。请在干净、干燥的台面上操作，别去拉扯纤细的传感器和电机导线。

## 2. 电源

把**电池盒**固定在板子底面，装入 **2 节 5 号（AA）电池**（约 3 V）。装配完成前先不要
打开**电源开关**。

## 3. 电机与车轮

把两个**直流减速电机**装在板上，使它们的输出轴在同一条直线上，然后把**车轮**分别压到
两根轴上。车头由一个小支撑/螺柱支起，形成稳定的三点支撑。

## 4. 安装 LM393

把 **LM393** 按正确方向插入芯片座，让第 1 脚（缺口或圆点）对准座上的标记，并确保
**每一只引脚都插进对应的孔**。引脚弯折或没插到位会使芯片短路，甚至烧毁。

## 5. 传感器

两个**光敏电阻**朝下装在车头、彼此靠近，使小车对中时两个都压在线上；小车一旦偏离，线就
从其中一个传感器下方滑走，小车便转回线上。传感器离地面的高度会影响灵敏度（见调试）。

## 6. 首次上电与调试

1. 把小车放到赛道上，使两个传感器都压在线上。
2. 打开开关。**指示灯 LED** 会显示每个比较器当前看到的情况。
3. 调节每个**微调电位器**，直到该侧能稳定区分黑线与白底（传感器越过边缘时 LED 干净地
   翻转）。
4. 让小车跑起来。若它在弯道处脱线，再微调电位器、传感器高度或线宽，直到循迹平滑。

## 使用与保养

传感器和电机的导线焊在板上，容易受力损坏，所以请轻拿轻放、保持干燥。长期不用时把电池
取出来，免得电池腐蚀电池盒。
