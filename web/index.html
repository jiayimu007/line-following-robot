Line-Following Robot Player

English·中文

The C++ engine in this repo simulates the analog car and exports its trajectory. This page is a small player: it loads that exported data and animates it on a canvas. It does not re-run the simulation — it only draws the precomputed frames.

Legend

  • Green dot: the sensor is on the line.
  • Red dot: the sensor has slipped off the line.
  • Both sensors on the line: the car drives straight. Only one on the line: it pivots back toward that side.

The motion shown here was computed by the C++ engine, not by this page. The real car has no microcontroller: the same on/off decision is made by an LM393 comparator in hardware, with no firmware. See the README for the circuit and the build.
